The following search modes are available.
0 - Get all metaphors
1 - Search metaphors by poem name
2 - Search metaphors by poet
3 - Search metaphors
4 - Search metaphors by source domain (synonyms supported)
5 - Search metaphors by target domain (synonyms supported)
6 - Search metaphors by any keyword
7 - Search metaphors by year range (Enter with this format :  <start_year>-<end_year>)
8 - Search metaphors with Gender

Enter -1 to exit.
